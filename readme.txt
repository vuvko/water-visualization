Управление камерой:
    W - движение вперёд по направлению взгляда
    S - движение назад
    A - движение налево (перпендикулярно вектору взгляда)
    D - движение вправо
    Pg Up - поднять камеру
    Pg Dw - опустить камеру
    Мышка (при зажатой левой кнопке) - поворот

Остальное управление:
    F1, F2, F3 - переключение между фиксированными позициями камер
    F5 - пауза симуляции
    F6 - воспроизведение симуляции
    Esc - выход
    ПКМ - добавление волн
